## Installation

1. Copy the `cybefox` folder to your AlphaCtyl `themes` directory
2. Update your AlphaCtyl config to use `cybefox` as the theme
3. Restart your AlphaCtyl instance

## Files Included

- `index.ejs` - Login page
- `dashboard.ejs` - Main dashboard
- `manage.ejs` - Server management
- `store.ejs` - Resource store
- `settings.ejs` - User settings
- `afk.ejs` - AFK rewards page
